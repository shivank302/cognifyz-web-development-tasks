# 🚀 Cognifyz Web Development Internship Tasks

This repository contains all 8 tasks completed as part of the **Cognifyz Web Development Internship Program**.
Each task demonstrates different concepts of front-end development including HTML, CSS, JavaScript, APIs, Forms, and Bootstrap.

---

## 📂 Project Structure

```
task-1 → Basic HTML Page
task-2 → Inline CSS Styling
task-3 → External CSS & Responsive Design
task-4 → JavaScript DOM Manipulation
task-5 → API Data Fetching
task-6 → Form Validation
task-7 → Bootstrap Responsive Website
task-8 → Sass Styling (UI Enhancement)
```

---

### 🔹 Task 1 – Basic Web Page
